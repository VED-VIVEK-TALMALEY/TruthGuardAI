# TruthGuard AI

**LLM Hallucination Detection, Verification & Correction Platform**

TruthGuard AI is a full-stack AI analytics platform designed to evaluate responses generated by multiple Large Language Models (LLMs), detect hallucinations, verify facts using trusted sources, allow manual corrections, and compare model performance through real-time, Worldometer-style analytics dashboards.

## Overview

The platform acts as an AI Response Diagnostic Tool that performs the following core pipeline for every submitted question:

1. **Question Input:** User submits a question via the UI.
2. **Multi-LLM Generation:** System queries GPT, Claude, Gemini, and LLaMA models.
3. **Evidence Retrieval:** Searches trusted sources (Wikipedia, Britannica, etc.).
4. **Fact Extraction:** Extracts key information from sources.
5. **Response Comparison:** Compares AI answers against factual evidence.
6. **Hallucination Detection:** Flags output based on similarity thresholds. 
7. **Score Calculation:** Computes final confidence, accuracy, and verification scores.
8. **Manual Review:** Optional human reviewer capability.
9. **Correction Engine:** Generates synthesized corrected answers.
10. **Global Analytics:** Live updating metrics mapped to Worldometer-style charts.

## Installation & Local Setup

### Prerequisites
- [Node.js](https://nodejs.org/) (v16.0 or higher recommended)
- Git

### 1. Clone the repository
```bash
git clone https://github.com/VED-VIVEK-TALMALEY/TruthGuardAI.git
cd TruthGuardAI
```

### 2. Setup Database & Backend Server
The backend uses Express and a local SQLite file (`database.sqlite`) which will automatically migrate the database schemas on launch.
```bash
cd backend
npm install
node server.js
```
The backend server will run on `http://localhost:5000`.

### 3. Setup Frontend Client
Open a **new terminal window**, and navigate to the frontend folder.
```bash
cd frontend
npm install
npm run dev
```
The UI dashboard will be accessible at `http://localhost:5173`.

## Features

- **Worldometer Inspired UI:** Dark aesthetic, large animated stat counters, and live Recharts representing global data.
- **Dynamic Scoring:** Multi-dimensional verification comparing AI confidence vs reality.
- **Comparison Engine:** Side-by-side diagnostic cards for multiple AI models simultaneously.

## Tech Stack
* **Frontend**: React, Vite, React Router, Recharts, Lucide-React, Vanilla CSS (Outfit Font).
* **Backend**: Node.js, Express, SQLite3.
